# Tableaux de bord de la LPHF

Cette application comprend 5 tableaux de bord pour visualiser les performances des équipes et des joueuses de la **Ligue professionnelle de hockey féminin** (LPHF).

Source des données : API HockeyTech ([référence](https://github.com/IsabelleLefebvre97/PWHL-Data-Reference))  
Récupération et traitement des données : repo [GitHub](https://github.com/Yu270/PWHL/tree/main/data)
